# ChessOx

High-performance Chess.com automation powered by the OxtaCore C++ engine and Selenium.

## Features
- **OxtaCore Engine:** Custom C++ chess engine for high-speed calculation.
- **Selenium Automation:** Real-time board detection and move execution on Chess.com.
- **TitanBrain:** Strategic layer for adaptive gameplay (Strategic Identity).

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Build the engine: `cmake -B build && cmake --build build --config Release`
3. Run the bot: `python titan_selenium.py`
